# RMS-WEB

A comprehensive web-based management system, integrating modern React frontends with a robust .NET backend. This component also includes elements of the system's migration from legacy WinForms to a modern web architecture.

## Components

- **Backend/RMS**: The core .NET 8 Web API and services.
- **Frontend/rms-react-frontend**: A primary React-based interface.
- **Frontend/rms-frontend**: Additional web interfaces.

## Features

- **Integrated Management**: Unified interface for various restaurant operations.
- **Legacy Integration**: Support for WinForms-based POS elements during migration.
- **Modern API**: Powered by .NET 8 with SignalR and Redis integration.
